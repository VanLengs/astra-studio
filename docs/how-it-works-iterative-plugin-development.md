# е·ҘдҪңжңәеҲ¶пјҡиҝӯд»ЈејҸжҸ’д»¶ејҖеҸ‘

жң¬ж–ҮжЎЈиҜҰз»ҶжЁЎжӢҹ Astra Studio дёӯдёҖж¬Ўе®Ңж•ҙзҡ„иҝӯд»ЈејҸжҸ’д»¶ејҖеҸ‘жөҒзЁӢгҖӮ

зӨәдҫӢйўҶеҹҹеҸӮиҖғ `course-workshop-plugins`пјҢеұ•зӨәд»Һ `init` еҲ°и§„еҲ’гҖҒжһ„е»әгҖҒж ЎйӘҢгҖҒеҸ‘еёғпјҢеҶҚеҲ°з¬¬дәҢиҪ®иҝӯд»ЈеҸҳжӣҙж—¶пјҢе·ҘдҪңеҢәжҳҜеҰӮдҪ•жј”еҢ–зҡ„гҖӮ

## ж–ҮжЎЈиҢғеӣҙ

жң¬ж–ҮйҮҚзӮ№иҜҙжҳҺпјҡ

- `studio/changes/` еңЁеҗ„йҳ¶ж®өеҰӮдҪ•еҸҳеҢ–
- еҹҹе·ҘдҪңеҢәдёҺжҸ’д»¶е·ҘдҪңеҢәзҡ„еҢәеҲ«
- `target_dir` еҰӮдҪ•дҪңдёәе®һзҺ°зҡ„еҚ•дёҖдәӢе®һжәҗ
- е·ІдәӨд»ҳзҡ„и®ҫи®Ўи®°еҪ•еҰӮдҪ•иҝӣе…Ҙ `studio/archive/`
- `create` дёҺ `modify` дёӨзұ»иҝӯд»Ји·Ҝеҫ„зҡ„е·®ејӮ

жң¬ж–Үд№ҹйҮҮз”ЁеҪ“еүҚе·Із»Ҹжҫ„жё…зҡ„дәӨдә’жЁЎеһӢпјҡ

- з”ЁжҲ·еҸӘеңЁе…ій”®йҳ¶ж®өиҫ№з•ҢиҝӣиЎҢзЎ®и®Ө
- зі»з»ҹиҮӘеҠЁжү§иЎҢйҳ¶ж®өеҶ…йғЁжӯҘйӘӨ
- `skill-creator` иҝҷзұ»еҶ…зҪ®иғҪеҠӣи§Ҷдёәзі»з»ҹеҶ…йғЁжү§иЎҢжӯҘйӘӨпјҢиҖҢдёҚжҳҜз”ЁжҲ·жүӢеҠЁиҝҗиЎҢзҡ„е‘Ҫд»Ө

## ж ёеҝғжЁЎеһӢ

### и®ҫи®ЎдёҺе®һзҺ°еҲҶзҰ»

- еҹҹзҹҘиҜҶжҳҜжҢҒз»ӯзҙҜз§Ҝзҡ„пјҢдҝқз•ҷеңЁ `studio/changes/{domain}`
- жҸ’д»¶е·ҘдҪңеҢәжҳҜдёҙж—¶зҡ„жҙ»и·ғеҸҳжӣҙи®°еҪ•
- е®һзҺ°е§Ӣз»ҲеӯҳеңЁдәҺ `target_dir`
- е·ІдәӨд»ҳзҡ„и®ҫи®Ўе·ҘдҪңеҢәиҝӣе…Ҙ `studio/archive/`
- з”ЁжҲ·иҙҹиҙЈйҳ¶ж®өзЎ®и®Ө
- зі»з»ҹиҙҹиҙЈйҳ¶ж®өжү§иЎҢ

иҝҷдёӘжЁЎеһӢзҡ„ж ёеҝғд»·еҖјеңЁдәҺпјҡ

- з”ЁжҲ·еҸӘеҲӨж–ӯж–№еҗ‘
- зі»з»ҹе®ҢжҲҗжһ„е»ә
- и®ҫи®ЎиҝҮзЁӢеҸҜиҝҪиёӘ
- е®һзҺ°е§Ӣз»ҲдҝқжҢҒе”ҜдёҖдәӢе®һжәҗ
